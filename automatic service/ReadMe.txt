========================================================================
    控制台应用程序：automatic service 项目概述
========================================================================

应用程序向导已为您创建了此 automatic service 应用程序。

本文件概要介绍组成 automatic service 应用程序的每个文件的内容。


automatic service.vcxproj
    这是使用应用程序向导生成的 VC++ 项目的主项目文件，其中包含生成该文件的 Visual C++ 的版本信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

automatic service.vcxproj.filters
    这是使用“应用程序向导”生成的 VC++ 项目筛选器文件。它包含有关项目文件与筛选器之间的关联信息。在 IDE 中，通过这种关联，在特定节点下以分组形式显示具有相似扩展名的文件。例如，“.cpp”文件与“源文件”筛选器关联。

automatic service.cpp
    这是主应用程序源文件。

/////////////////////////////////////////////////////////////////////////////
其他标准文件:

StdAfx.h, StdAfx.cpp
    这些文件用于生成名为 automatic service.pch 的预编译头 (PCH) 文件和名为 StdAfx.obj 的预编译类型文件。

/////////////////////////////////////////////////////////////////////////////
其他注释:
obj.h, obj.cpp
    这个类用于托管类，每个继承该类的对象在一桢执行结束后会被自动释放掉,手动调用retain会继续保持，但在不需要使用的时候必须调用release才会释放。
Date.h, Date.cpp
    这个类用于日期时间管理，默认不带参数的构造函数的创建当前时间，带参构造函数为传入参数顺序为 时 分 秒
LOG.h, LOG.cpp
    这个类用于记录系统操作日志，保存到LOG文本，初始化时需要调用init，记录日志时调用RECORD，参数为需要保存的消息日志，会自动加入当前时间。
task.h, task.cpp
    这个类用于创建任务，也可直接调用task_management::create(),传入参数为 任务执行时间（每日或者每周固定日期），bat命令脚本，执行时间，任务名称。

应用程序向导使用“TODO:”注释来指示应添加或自定义的源代码部分。
